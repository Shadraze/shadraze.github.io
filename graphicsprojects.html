Graphics Projects

Matter Construct: 3D Model Drawing Application

Matter Construct is my undergraduate degree's 6th semester's project. The program allows a user to draw in 3d space and generate 3d models with it. These models can be exported to a standard .fbx format which allows other 3d programs to access it. In the demo, I use the default 3D viewer that comes with Windows and show the wireframe beneath generated by the program. Through Matter Construct, I became famialirized with 3d curves and also a lot of newer mathematical concepts. It proved to be a difficult but fun journey.

Wireframe Renderer in WinBGIm

A small program that accepts .obj files and renders the wireframe of the 3D model contained in the file. Although, it has no practical applications, it was a learning experience and was quite fun to see 3d models I created be displayed in a program meant for basic 2d graphics.

Admittedly though, it can't handle more complex models as seen in the second model. WinBGIm being a software renderer is more of a learning tool and it is also quite an old piece of software.
